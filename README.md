# 📊IPL-capstone-project
## 📌 Overview
This project is a comprehensive data analysis and visualization study of the Indian Premier League (IPL) using Python. The dataset includes match-level and team-level performance data across multiple IPL seasons.

The project focuses on extracting meaningful insights from cricket data through exploratory data analysis (EDA) and visual storytelling using data visualization libraries such as Matplotlib and Seaborn.

Through structured analysis and graphical representation, this project highlights trends in team performance, match outcomes, player impact, and scoring patterns in the IPL.

## 🎯 Objective

Matplotlib

Seaborn

## 📈 Key Insights
- 1️⃣ Toss Decision Trends

Analysis of toss decisions shows that the majority of teams prefer fielding first after winning the toss.
- 2️⃣ Toss Winner vs Match Winner Correlation

The analysis indicates that winning the toss does not always guarantee winning the match, but there is a moderate positive correlation between toss winners and match winners
- 3️⃣ Top Scorers Have High Match Impact

Top individual scorers across seasons consistently contribute significantly to team victories.

Matches where a player scored above a high threshold (e.g., 70+ runs) showed increased win probability for their team.
